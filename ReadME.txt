USER STORIES 

User story 1 :

LE client désire une wishlist pour tous ses amis avec leur noms, prénoms, jour et mois de naissance dans le but de  visualiser sa liste d'amis.

User story 2 :

IL veux que la liste de ses amis soit triée suivant les mois et les jours de ses amis.

User story 3 :

LE client souhaite accéder ?sa liste d'amis grace ?une interface graphique.

User story 4 :

Le client peut  ajouter des idées cadeaux et peut  les afficher triées .
